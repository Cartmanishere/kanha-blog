# Kanha Blog

Personal blog built with Hugo and the Terminal theme.

## Quick Start

```bash
# Run locally
hugo server

# Build site
hugo
```

## Structure

- `content/posts/` - Blog posts
- `layouts/` - Custom templates and shortcodes
- `assets/css/` - Custom CSS
- `themes/terminal/` - Terminal theme (git submodule)

## Adding Posts

Create `.md` files in `content/posts/` with front matter:

```yaml
---
title: "Post Title"
date: YYYY-MM-DD
---
```
